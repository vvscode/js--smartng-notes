Реализовать директиву map в модуле adnis. Директива принимает два параметра:

reports. Отчеты, которые нужно отображать в виде маркеров на карте. Каждый report имеет формат вида
{
  location: {
    latitude: 5.877674994795961,
    longitude: 38.45339266561527
  }
}
boundingRect. Описывает положение карты. (в Leaflet setView). Имеет формат:
{
  latitude: 51.505,
  longitude: -0.09,
  zoom: 13
}
Ваша директива должна иметь следующую функциональность:

при замене объекта reports все маркеры с карты должны исчезать и появляться новые
при изменении boundingRect карта изменяет свою позицию
при перетаскивании карты необходимо изменять данные в boundingRect
Для отображения карты используйте следующий слой с данными:

L.tileLayer(
  'http://otile1.mqcdn.com/tiles/1.0.0/map/{z}/{x}/{y}.png',
  {
    id: 'examples.map-9ijuk24y'
  }
).addTo(map);
